# Simulador-de-RPG

Este é um projeto final para a disciplina de Desenvolvimento Web Front-End, focado em criar uma aplicação web interativa e responsiva que simula uma batalha simples no estilo RPG (Role-Playing Game).

# Objetivo do Projeto
O objetivo principal é demonstrar o domínio das tecnologias HTML5, CSS3 e JavaScript através da construção de um Simulador de Batalha interativo, onde o usuário cria um herói e se engaja em um combate simples baseado em turnos contra um monstro.
O projeto cumpre todos os requisitos de estruturação semântica, layout responsivo e lógica de programação exigidos.
